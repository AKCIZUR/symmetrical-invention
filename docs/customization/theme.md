# Téma

Tady se mění základní vzhled dokumentace.

## Co lze upravit

- barvy
- fonty
- tmavý a světlý režim
- layout hlavičky
- vzhled banneru
- šířku obsahu

## Jak postupovat

Nejdřív změň `mkdocs.yml`, potom doplň vlastní CSS do `docs/assets/css/extra.css`.

## Doporučení

Nezasahuj do balíčku tématu v `site-packages`. Vlastní design řeš přes overrides a CSS soubory v repu.
