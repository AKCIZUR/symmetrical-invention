# Barvy

Výběr barev ovlivňuje čitelnost, kontrast i celkový dojem z dokumentace.

## Primární a akcentní barva

Primární barva je vhodná pro hlavičku a klíčové prvky. Akcentní barva by měla sloužit pro tlačítka, odkazy a zvýraznění.

## Tmavý režim

Tmavý režim je vhodné držet jako default pro technickou dokumentaci. Pokud web cílí i na širokou veřejnost, může být lepší přepínač a respektování systémového nastavení.

## Pravidlo konzistence

Na webu by se neměly mísit příliš odlišné odstíny stejné role. Lepší je menší, ale stabilní paleta.
