# Barvy

Barevnost je nejlepší řešit přes CSS proměnné.

## Příklad

```css
:root {
  --md-primary-fg-color: #334155;
  --md-accent-fg-color: #6366f1;
}
```

## Doporučení

Používej jednu hlavní barvu a jednu akcentovou. Přehnaně barevné dokumentace se čtou hůř.

## Osvědčená kombinace

- primární: slate
- akcent: indigo
- upozornění: amber
- success: emerald
