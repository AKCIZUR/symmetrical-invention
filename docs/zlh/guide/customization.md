# Customization

## Theme overrides

Use `docs/overrides/` to change layout elements such as:

- head metadata
- breadcrumbs
- header content
- footer content
- search UI styling

## Visual system

The starter ships with:

- stronger code block framing
- compact hero cards
- responsive grids
- darker default palette with a light-mode toggle

## Safe extension pattern

Prefer CSS variables and template overrides before editing HTML content directly.
