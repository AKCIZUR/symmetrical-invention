# Theme

## Enabled features

The starter enables:

- breadcrumbs via `navigation.path`
- section landing pages via `navigation.indexes`
- footer navigation via `navigation.footer`
- active TOC highlighting via `toc.follow`
- search highlighting and sharing via `search.highlight` and `search.share`

## Why this matters

These features cover the practical documentation UX users expect on a modern static site.
