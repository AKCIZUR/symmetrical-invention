# Pluginy

## Build spadne po přidání pluginu

Zkontroluj, jestli je plugin v `requirements.txt` a jestli má správné nastavení v `mkdocs.yml`.

## Galerie nefunguje

Ověř, že `glightbox` je nainstalovaný a že obrázky jsou skutečně v dokumentaci načtené jako Markdown image.

## Git metadata se nezobrazují

Ujisti se, že repozitář má git historii a že běží build v prostředí, které vidí commit log.

## Hledání nic nenajde

Zkrať názvy nadpisů, přidej konkrétní termíny a zkontroluj, že `search` plugin zůstává aktivní.
