# Téma

Tady se mění vzhled webu bez zásahu do zdrojového kódu MaterialX.

## Barvy

Barvy se upravují přes palette v `mkdocs.yml` a přes CSS proměnné v `extra.css`. Tím lze vytvořit tmavý, světlý nebo výrazně brandový styl.

## Fonty

Pro hlavní i monospace text lze nastavit jiné fonty a sladit tak dokumentaci s vizuální identitou webu.

## Rozvržení

Šířka obsahu, hlavička, sidebar i footer se dají upravit pomocí vlastního CSS a šablon v `overrides/`.
