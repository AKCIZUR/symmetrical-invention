# Slovník

Krátký slovník pojmů použitých v dokumentaci.

## Markdown

Textový formát pro psaní dokumentace pomocí jednoduché značkovací syntaxe.

## Override

Vlastní šablona, která přepisuje vybranou část vzhledu bez zásahu do zdrojového kódu tématu.

## TOC

Table of contents, tedy obsah stránky nebo lokální navigace k nadpisům.

## Breadcrumbs

Navigační cesta ukazující, kde se uživatel v hierarchii dokumentace právě nachází.
