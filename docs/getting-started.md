# Rychlý start

## Instalace

```bash
pip install -r requirements.txt
```

## Lokální spuštění

```bash
mkdocs serve
```

## Produkční build

```bash
mkdocs build --strict
```

## Nasazení

Po pushi do `main` se repozitář publikuje přes GitHub Actions.

## Doporučený postup

1. uprav `site_url`
2. uprav `repo_url`
3. nahraď logo a favicon
4. doplň vlastní obsah
