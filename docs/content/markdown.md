# Markdown

Markdown je základní formát celé dokumentace.

## Co podporovat

- nadpisy
- seznamy
- tabulky
- bloky kódu
- admonitions
- záložky
- footnotes
- obrázky

## Dobrá praxe

Piš krátké odstavce. Čtenář dokumentace většinou nehledá literární text, ale rychlou odpověď.

## Příklad

```markdown
!!! tip
    Krátká rada, která pomůže hned.
```
