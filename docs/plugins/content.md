# Obsah a include

Tato sekce připravuje prostor pro opakovaně použitelný obsah a budoucí rozšíření.

## Co se v core verzi používá

- standardní Markdown
- tabulky
- admonitions
- taby
- kódové bloky
- bloky s poznámkami

## Pro budoucí rozšíření

Sem můžeš později doplnit:

- include markdown
- snippet systémy
- generované části z YAML nebo JSON
- šablonování přes makra

## Doporučení

Když opakuješ stejný text na více místech, vyplatí se ho vytáhnout do samostatného souboru a zahrnout do dokumentace jedním zdrojem pravdy.
