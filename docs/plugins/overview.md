# Přehled pluginů

Tahle core verze používá jen stabilní pluginy, které mají jasný přínos pro produkční dokumentaci.

## Aktivní pluginy

- `search` – fulltextové hledání
- `git-revision-date-localized` – datum poslední změny a vytvoření stránky
- `git-authors` – autoři stránky z Git historie
- `glightbox` – lightbox pro obrázky
- `minify` – menší HTML výstup

## Proč právě tyto

Cíl je mít rychlý, stabilní a čitelný web bez zbytečné složitosti.

!!! note
    Rozšiřující pluginy je lepší přidávat až po ověření, že core build prochází bez chyb.
