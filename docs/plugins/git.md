# Git metadata

Tahle část ukazuje, proč jsou Git pluginy v dokumentaci užitečné.

## Zapnuté pluginy

- `git-revision-date-localized`
- `git-authors`

## Co přinášejí

- poslední aktualizace stránky
- datum vytvoření nebo první publikace
- seznam autorů z Git historie

## Proč je to důležité

Čtenář hned vidí, zda je obsah aktuální. U technické dokumentace je čerstvost často stejně důležitá jako samotný obsah.

## Doporučení

Zobrazuj metadata konzistentně na všech stránkách a nepřidávej příliš mnoho rušivých údajů.
