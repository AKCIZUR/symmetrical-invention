# FAQ k pluginům

## Který plugin je nejdůležitější?

Pro běžný produkční web jsou klíčové `search`, `git-revision-date-localized`, `git-authors`, `glightbox` a `minify`.

## Co přidat později?

Až bude core stabilní, můžeš doplnit i pluginy jako blog, RSS, redirects, macros nebo auto-generovanou navigaci.

## Co nedělat hned na začátku?

Nedávej do core verze příliš mnoho pluginů najednou. Každý další plugin znamená další bod možného selhání buildu.
