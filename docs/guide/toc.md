# Obsah stránky

Obsah stránky se generuje z nadpisů v Markdownu a pomáhá čtenáři rychle přeskočit na správnou část.

## Výhody

- automatické vytvoření z nadpisů
- aktivní zvýraznění aktuální sekce
- rychlá orientace na delších stránkách

## Jak psát nadpisy

Používej konzistentní hierarchii:

```markdown
# Hlavní téma
## Podtéma
### Detail
```

## Dobrá praxe

Nezalamuj obsah do příliš mnoha úrovní. Na dokumentačním webu obvykle stačí do třetí úrovně.
