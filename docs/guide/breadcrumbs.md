# Breadcrumbs

Breadcrumbs ukazují uživateli, kde se právě nachází v rámci webu.

## Přínos

- lepší orientace
- rychlý návrat o úroveň zpět
- přehledná hierarchie dokumentace

## Příklad

```text
Domů > Průvodce > Navigace > Breadcrumbs
```

## Jak je používat

Breadcrumbs jsou nejpřínosnější u hlubší struktury dokumentů. Když má web více úrovní, výrazně zlepšují čitelnost.
