# Navigace

Stránka popisuje, jak je web rozdělený a jak se v něm uživatel orientuje.

## Sidebar

Sidebar má být zrcadlo struktury dokumentů. Hlavní sekce jsou krátké, podsekce tematicky čisté a názvy musí odpovídat obsahu souborů.

## Breadcrumbs

Breadcrumbs ukazují aktuální cestu dokumentem. Pomáhají hlavně u hlubších struktur, kde uživatel přeskakuje mezi sekcemi.

## Předchozí a další stránka

Na konci článků se doporučuje přidat navigaci předchozí/další, aby čtenář mohl projít logickým pořadím témat bez návratu do menu.

## On this page

Lokální TOC u dlouhých stránek pomáhá rychle přeskočit k příslušnému nadpisu. Aktivní sekce se zvýrazňuje při scrollování.
