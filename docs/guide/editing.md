# Editace dokumentace

Každá stránka je přímo propojená s GitHubem. Díky tomu lze obsah upravit bez složité správy v nějakém externím CMS.

## Doporučený workflow

1. Otevři dokument v repozitáři.
2. Klikni na **Editovat tuto stránku**.
3. Uprav Markdown.
4. Otevři pull request nebo commitni změnu.
5. GitHub Actions automaticky přegeneruje web.

## Výhody

- jednoduchá kontrola verzí
- snadný review
- historický přehled změn
- rychlé nasazení

## Kdy je to vhodné

- dokumentace pro vývojáře
- technické návody
- interní wiki
- otevřené komunitní projekty

!!! warning
    Pokud používáš více editorů nebo větší tým, vyplatí se zavést jednotné názvy souborů a krátký stylový manuál.
