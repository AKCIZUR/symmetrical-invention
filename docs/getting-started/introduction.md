# Úvod

MaterialX je vzhled a sada funkcí pro MkDocs, které z dokumentace dělají přehledný a moderní web. Hodí se pro projektovou dokumentaci, interní znalostní bázi i veřejné produktové návody.

## Proč ho použít

- rychlý statický web
- jednoduchá správa přes Markdown
- čisté rozložení pro čtení i hledání
- výborný základ pro GitHub Pages

## Pro koho je tento projekt

- frontend vývojáře
- technické autory
- týmy, které chtějí dokumentaci spravovat v GitHubu
- projekty, kde je důležitý přehled, rychlost a nízká údržba

## Jak funguje

MkDocs přečte Markdown soubory, zkonvertuje je do HTML a vygeneruje statický web. MaterialX se postará o vzhled, navigaci a lepší uživatelskou zkušenost.

!!! note
    Tento projekt je připravený tak, aby šel dál rozšiřovat bez zásahu do instalovaného balíčku tématu.
