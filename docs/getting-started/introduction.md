# Úvod

Tato sekce vysvětluje, k čemu dokumentace slouží a jaký má cíl.

## Smysl dokumentace

Dokumentace má být zdroj pravdy pro celý web nebo produkt. Uživatel v ní najde návod, editaci obsahu, technické fungování i řešení problémů.

## Princip práce s Markdownem

Každý článek je samostatný `.md` soubor. Díky tomu se dobře verzují v Gitu, snadno upravují v editoru a automaticky se promítají do generované stránky.
