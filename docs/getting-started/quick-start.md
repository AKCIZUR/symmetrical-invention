# Rychlý start

Tahle stránka slouží jako nejkratší možná cesta k prvnímu funkčnímu webu.

## Co udělat hned

Otevři `mkdocs.yml`, doplň vlastní `site_url` a `repo_url`, pak uprav úvodní stránku `docs/index.md` a spusť lokální náhled.

## První obsah

Začni jedním úvodním článkem, jedním průvodcem a jednou referencí. Web pak může růst po menších blocích bez chaosu v navigaci.

## První nasazení

Jakmile stránka vypadá správně lokálně, pusť workflow v GitHub Actions a publikuj na GitHub Pages.
