# Rychlý start

Tato sekce je pro okamžité spuštění projektu.

## 1. Spusť lokální server

```bash
mkdocs serve
```

## 2. Otevři web

V prohlížeči otevři adresu, kterou vypíše konzole, obvykle:

```text
http://127.0.0.1:8000/
```

## 3. Uprav první stránku

Začni v souboru `docs/index.md`.

## 4. Přidej další stránky

Stačí vytvořit nový `.md` soubor a přidat ho do `nav` v `mkdocs.yml`.

## Minimální ukázka

```markdown
# Nová stránka

Krátký úvodní text.

## Sekce

Další obsah.
```
