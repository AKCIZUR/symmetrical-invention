# Instalace

Zde je základní postup, jak si projekt připravit lokálně.

## Požadavky

Potřebuješ Python 3.12 nebo novější, Git a prostředí pro práci s Markdown soubory. Pro lokální náhled se používá `mkdocs serve`.

## Instalace závislostí

Vytvoř virtuální prostředí, aktivuj ho a nainstaluj balíčky z `requirements.txt`. Tím získáš MkDocs, MaterialX a potřebné pluginy.

## Kontrola instalace

Spusť `mkdocs build --strict`. Pokud projde bez chyb, je projekt připravený pro GitHub Pages i lokální testování.
