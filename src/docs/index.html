Jak je web postavený

Tahle sekce vysvětluje fungování webu, strukturu souborů a způsob, jak rychle přidávat nové stránky.

Struktura projektu

Projekt je rozdělený na zdrojovou část, statický build a workflow pro deploy. Každá část má jasnou roli a dá se upravovat samostatně.

src/
  index.html
  docs/
  assets/
  data/

scripts/
  build.mjs

dist/
  ...vygenerovaný web

Jak funguje

Stránky se renderují jako hotové HTML. JavaScript doplňuje menu, vyhledávání, TOC a přechody mezi stránkami. Díky tomu je web rychlý a stabilní.

Editace

Obsah se upravuje přímo v HTML souborech. Pro nové stránky stačí zkopírovat existující šablonu a přidat ji do navigace v assets/js/app.js.

Deploy

GitHub Actions spouští build, vytváří dist/ a publikuje výstup do větve gh-pages.
