Dokumentace

UI pravidla

Shadcn dark B&W styl

Vizuál drží černou, bílou a šedou. Barevné jsou jen hrany code bloků, aby zůstala dokumentace čistá.

Paleta

Primární vrstvy jsou téměř monochromní. Glass panely používají průhlednost, jemný blur a tenký border.

:root {
  --bg: #050505;
  --surface: rgba(18, 18, 18, .72);
  --border: rgba(255, 255, 255, .08);
  --text: #f5f5f5;
}

Typografie

Text je čitelný, kompaktní a bez zbytečně velkých mezer. Nadpisy jsou výrazné, ale stále minimalistické.

Liquid glass

Sklo tvoří tenká linka, blur a velmi jemný stín. Panely nepůsobí jako lesklé objekty, ale jako matná průhledná vrstva.
