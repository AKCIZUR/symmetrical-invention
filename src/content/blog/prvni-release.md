# První release

Tahle verze je postavená jako čistý start pro dokumentační web.

## Záměr

- jednoduchý build
- statický deploy
- dobrá čitelnost
- možnost rozšiřování

## Co je hotové

- homepage
- docs stránky
- blog
- search paleta
- megamenu
- page transitions

## Další krok

Doplnit širší sadu komponent, example pages a případně i MDX bloky.
