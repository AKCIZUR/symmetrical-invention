# Proč Astro

Astro je vhodný, když chceš:

- statický web
- čistý Markdown content
- moderní page transitions
- možnost přidat interaktivitu jen tam, kde je potřeba

## Co je silná stránka

Obsah a UI jsou oddělené. To je přesně to, co chce dokumentace potřebovat.

## Kdy je to nejlepší volba

Když je důležitý design, rychlost a jednoduché nasazení na GitHub Pages.
