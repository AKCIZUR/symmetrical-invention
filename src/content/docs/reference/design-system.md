# Design systém

## Principy

- monochromní barevnost
- glass vrstvy
- měkký radius
- žádné agresivní barvy
- důraz na typografii a spacing

## Komponenty

- tlačítka
- karty
- menu
- dialog
- TOC
- breadcrumbs
- code block

## Tokeny
