# Code bloky

## Cíl

Kódové bloky mají být jednoduché, čitelné a bez přehnaného designu.

## Barevná linka podle jazyka

- HTML → modrá
- CSS → oranžová
- JS → žlutá
- TS → cyan

## Styl

- tenká levá linka
- tmavé pozadí
- jemná bordura
- zaoblení
- malý horní panel

## Příklad

```html
<div class="card">Ahoj</div>
```

```css
.card {
  backdrop-filter: blur(24px);
}
```

```js
console.log('NoirDocs');
```
