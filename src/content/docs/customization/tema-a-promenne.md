# Téma a proměnné

## Základ

Veškerý vzhled je řízený přes CSS proměnné a Tailwind utility třídy.

## Co měnit

- barvu pozadí
- border opacity
- shadow intensity
- radius komponent
- spacing
- typografii

## Doporučený postup

Nejprve měň globální proměnné. Až potom sahat do jednotlivých komponent.

## Minimalismus

Drž vše na třech vrstvách:
1. základní pozadí
2. glass plocha
3. zvýrazněný hover stav
