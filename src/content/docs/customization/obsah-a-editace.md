# Obsah a editace

## Přidání stránky

1. vytvoř nový `.md` soubor v `src/content/docs/`
2. doplň frontmatter
3. přidej záznam do navigace
4. spusť build

## Frontmatter

Minimálně:

- title
- description
- section
- order

## Blog

Blogové články patří do `src/content/blog/` a používají datum publikace.

## Editace přes GitHub

Každý soubor můžeš upravit přes GitHub web, pokud máš otevřený repozitář.
