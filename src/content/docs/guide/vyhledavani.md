# Vyhledávání

## Jak funguje

Paleta hledání běží na klientovi a filtruje připravený index stránek.

## Ovládání

- `Ctrl + K`
- `⌘ + K`
- klik na tlačítko „Vyhledat“

## Výhody

- okamžité výsledky
- bez serveru
- funguje i na GitHub Pages
- vhodné pro dokumentaci i blog

## Důležité

Hledání je navržené pro malé a střední dokumentační weby.
